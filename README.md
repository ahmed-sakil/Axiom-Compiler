ğŸ“ Axiom Syntax Guide

Axiom is designed to be C-like but simplified.

1. Variables & Math

// Define variables using 'set'
set width = 10;
set height = 25;

// Axiom handles complex math order-of-operations

// Division and Subtraction
set result = area / 5 - 2;


2. Output

// Print results to the console
print result;


3. Comments

// Lines starting with double-slashes are ignored
